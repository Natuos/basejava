#第二章
##1.JDK、JRE、JVM的关系
* JDK = JRE + java开发工具
* JRE = JVM + 核心类库
##2.环境变量path配置及其作用
* 环境变量的作用是为了在DOS的任意目录，可以去使用java和javac命令
* 先配置JAVA_HOME = 指向jdk安装的主目录
##3. Java编写步骤
* 编写java的源代码
* javac 编译，得到对应的 .class 字节码文件
* java 运行，本质就是把 .class 加载到jvm 运行
